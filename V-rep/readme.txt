## Get started

- Copy this file folder inside the ROS workspace source folder
- Make a catkin_make
- Digit "rosrun baxter publisher_ROS_VREP" and press enter
- Run CoppeliaSim
- In file/openScene/Computer find the folder in the ROS workspace
- in this folder, open the subfolder baxter and open the .ttt file
- once the scene is loaded go back to the shell and digit "rosrun baxter logger" and press enter
- when all is set start the simulation
